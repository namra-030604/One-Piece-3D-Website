# One Piece 3D Arc Explorer

A cinematic portfolio website built with Three.js, GSAP, WebGL. Click to journey through each One Piece arc in 3D.

## How to Use

Open `index.html` in any modern browser. No build step, no server, no dependencies to install — everything loads from CDN.

## Controls

- **Click / Tap** — Navigate to the next arc
- **Space / Arrow Right** — Next arc
- **Arrow Left** — Previous arc

## Arcs

1. **Set Sail** — The Grand Line awaits
2. **East Blue** — Where it all began
3. **Alabasta** — Loyalty tested in the sands
4. **Skypiea** — Above the clouds lies a golden city
5. **Water 7** — A crew torn apart
6. **Marineford** — The war that changed the world
7. **Dressrosa** — Beneath the petals, a dictator's chains
8. **Wano** — An isolated nation under the shogun's shadow

After the final arc, a credits screen appears. Click again to loop back to the beginning.

## Tech Stack

- **Three.js r128** — 3D WebGL rendering (CDN)
- **GSAP 3.12** — Camera animations and UI transitions (CDN)
- **UnrealBloomPass + FilmPass** — Post-processing effects (CDN)
- **Google Fonts** — Pirata One + Cinzel (CDN)
