One Piece 3D Arc Explorer — Portfolio Website
1. Executive Summary
Project Name: One Piece 3D Arc Explorer
Type: Personal Fan Site / Portfolio Project
Platform: Replit (Vanilla HTML + CSS + JavaScript)
Owner: Personal (Portfolio Showcase)
Target Completion: ASAP
Version: 1.0

This is a fully frontend, single-page 3D animated website inspired by the One Piece anime. The site uses a click-to-zoom cinematic navigation model, where each click takes the user deeper into a new One Piece arc — rendered in a unique 3D Three.js scene. The site is built to impress recruiters with advanced frontend animation skills.

2. Problem Statement
Portfolio websites for developers are often static, boring, and forgettable. This project solves that by creating a visually immersive, recruiter-stopping experience that demonstrates mastery of:

3D WebGL rendering (Three.js)

Cinematic camera animation (GSAP + lerp)

Post-processing effects (Bloom, Film Grain)

Scroll/click-driven storytelling

UI layering over 3D canvas

6. Feature Requirements
6.1 Must Have (MVP)
ID	Feature	Description
F01	3D Canvas Background	Full-screen Three.js WebGL canvas behind all UI, position: fixed, z-index: 0
F02	Click-to-Zoom Navigation	Click anywhere → camera zooms forward via GSAP tween transitioning to next arc scene
F03	Arc Scene Switcher	8 distinct Three.js scenes (Intro + 7 arcs), each with unique geometry, colors, and lighting
F04	Mouse Parallax	Camera position/rotation lerps toward cursor for a floating, cinematic feel
F05	Bloom / Glow Effect	EffectComposer + UnrealBloomPass for glowing edges and lights
F06	Film Grain Overlay	FilmPass or custom shader for cinematic noise texture
F07	Arc Title Overlay	HTML/CSS arc name + one-line description layered over canvas using z-index: 10
F08	Click Prompt	Subtle pulsing "Click to Continue" text at the bottom
F09	Idle Animation	Floating/bobbing position.y sine wave + slow Y-axis rotation on main 3D objects
F10	Responsive Canvas	Canvas resizes on window resize, camera aspect ratio updates
6.2 Nice to Have (Post-MVP)
ID	Feature	Description
F11	Arc Progress Indicator	Small dots or arc name breadcrumb showing current position (e.g. ● ○ ○ ○)
F12	Ambient Sound	Subtle arc-themed ambient audio toggle (ocean waves, wind, etc.)

8. 3D & Animation Technical Specification
Based on the Solais website effect analysis:
​

8.1 Core Rendering Stack
text
Three.js r128+
├── WebGLRenderer (antialias: true, alpha: true)
├── PerspectiveCamera (FOV: 75)
├── EffectComposer Pipeline
│   ├── RenderPass (base scene)
│   ├── UnrealBloomPass (strength: 1.2, threshold: 0.2, radius: 0.8)
│   └── FilmPass (noise intensity: 0.25, scanlines: false)
└── requestAnimationFrame render loop
8.2 Camera Zoom Transition (Click Navigation)
javascript
// On click → GSAP tween camera forward, then swap scene
gsap.to(camera.position, {
  z: camera.position.z - 30,   // zoom into scene
  duration: 1.4,
  ease: "power3.inOut",
  onComplete: () => loadNextArcScene()
});
8.3 Mouse Parallax
javascript
// Lerp camera toward mouse — smooth floating effect
camera.position.x += (targetX - camera.position.x) * 0.05;
camera.position.y += (targetY - camera.position.y) * 0.05;
camera.lookAt(scene.position);
8.4 Idle Animation Loop
javascript
// Applied to primary 3D object in each scene
mesh.rotation.y += 0.003;
mesh.position.y = Math.sin(clock.getElapsedTime() * 0.8) * 0.4;
8.5 UI Layer Stack
text
z-index: 100  →  Click prompt ("Click to Continue")
z-index: 10   →  Arc title + description (HTML/CSS)
z-index: 5    →  Progress dots (arc indicator)
z-index: 0    →  Three.js canvas (position: fixed)
