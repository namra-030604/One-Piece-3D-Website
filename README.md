# One Piece 3D Arc Explorer

Cinematic 3D portfolio website inspired by the One Piece anime.

Built with:
- Three.js
- GSAP
- WebGL
- TypeScript
- Vite

## How to run locally

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```
